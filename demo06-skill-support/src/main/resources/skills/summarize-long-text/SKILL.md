# 长文摘要 Skill

按以下结构输出摘要：

## TL;DR
**一句话**说清整篇文章在讲什么。

## 核心要点
3-7 个要点，每点一句话。按重要性排序。

## 关键数据/引用（如果原文有）
- 重要数字、统计、引述
- 保留原文用词，不要改写

## 个人评估（可选）
仅当用户明确要求时才提供你对内容的看法。默认只摘要不评论。

## 长度规则
- 原文 < 1000 字：要点 3 个左右
- 原文 1000-5000 字：要点 5 个左右
- 原文 > 5000 字：要点 7 个左右，可以再加"按章节分述"

## 注意
- 不要在摘要里编造原文没有的信息
- 如果原文有事实错误，可以在最后用 ⚠️ 标注，但不要修改摘要内容
